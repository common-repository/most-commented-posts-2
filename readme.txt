Plugin to show the image of most commented posts

== Description ==

With that plugin you can show images of most commented posts as a widget on your blog or you can also call the PHP function directy and get most commented posts images to show it every where you want.

== Installation ==

Follow the below steps to intall the plugin:
1- Send plugins's folder to plugins path, it should be /wp-content/plugins/
2- Access the blog panel and activate the plugin
3- Now, in the Appearance area, clique on Widget and right there you will be able to activate the widget of this plugin
4- Change the options you want and click on button Update
